1. Project Title / Headline

Apollo Hospitals: Patient Leads Flow Dashboard
An interactive Power BI dashboard built to visualize patient flow, bed occupancy, diagnosis trends, doctor feedback volume, and billing insights for hospital performance analysis.

2. Short Description / Purpose

This Power BI report enables users to monitor patient admissions, discharge patterns, resource usage, and revenue performance in a healthcare setting, supporting data-driven decisions for hospital administrators and operations teams.
